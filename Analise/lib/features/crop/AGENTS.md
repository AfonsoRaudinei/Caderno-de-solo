# Agente Feature Crop

## Escopo

Use para widgets e fluxos relacionados a cultura/crop quando estiverem neste modulo.

## Regras

- Mantenha selecao de cultura tipada e consistente com dados de `culturas`/referencias.
- Evite duplicar datasets; procure fonte existente antes de criar nova lista.
- UI deve ser leve e reaproveitar componentes comuns.
- Nao altere dados, textos, campos ou arquivos que nao tenham sido citados no prompt ou no pedido atual.

## Conclusao

Use sempre o checklist final obrigatorio do agente mestre.
