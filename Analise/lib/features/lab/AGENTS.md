# Agente Feature Lab

## Escopo

Use para calibracao, telas e fluxos laboratoriais que ainda estejam no modulo `lab`.

## Regras

- Verifique se a implementacao correta esta em `lab` ou `laboratorio`; nao duplique fluxo entre os dois modulos.
- Calibracoes devem preservar fonte, perfil e parametros usados.
- Alteracoes de recomendacao/calibracao precisam de teste de dominio quando envolverem calculo.
- Nao altere dados, textos, campos ou arquivos que nao tenham sido citados no prompt ou no pedido atual.

## Conclusao

Use sempre o checklist final obrigatorio do agente mestre.
