# Agente Feature Culturas

## Escopo

Use para selecao de culturas, cultivares, tecnologias, autores/referencias e visualizacoes de dados agronomicos.

## Regras

- Dados de cultura devem ter fonte clara e nao serem duplicados sem necessidade.
- Se alterar tabela/referencia agronomica, documente premissa e adicione teste quando houver calculo.
- Providers devem separar filtro/seleção de transformacoes de dados.
- Listas grandes devem usar builders e filtros eficientes.
- Nao altere dados, textos, campos ou arquivos que nao tenham sido citados no prompt ou no pedido atual.

## Conclusao

Use sempre o checklist final obrigatorio do agente mestre.
